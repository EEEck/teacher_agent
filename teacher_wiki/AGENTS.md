# KlassenPilot Memory Rules

This wiki is the persistent memory for KlassenPilot.

## Editing Rules
- Never auto-write changes from the model directly.
- Always show proposed diffs/updates first.
- Persist changes only after explicit teacher approval.
- Keep entries concise, factual, and timestamped where useful.
- Use pseudonymous student IDs (`S-001`, `S-002`, ...); no real names.
